# java-project

This is a Spring Boot project created to practice JPA and Hibernate.

Tech Stack
•Java 17
•Spring Boot
•Spring Data JPA
•Hibernate
•MySQL
•Maven

Features
•Save user details using JPA Repository
•Basic CRUD operations
•Spring Boot auto-configuration

How to Run
•Import project into STS
•Configure database in application.properties
•Run BootjpaexampleApplication
